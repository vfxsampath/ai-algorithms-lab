# Recommendation-System Algorithms

Recommendation systems rank and suggest relevant items based on item attributes, user behaviour, or both.

| Algorithm | Application | Status |
|---|---|---|
| [Content-Based Filtering](content_based_filtering/) | Attribute-based personalized recommendation | Complete |
| Collaborative Filtering | User and item interaction patterns | Next |
| Matrix Factorization | Latent user-item preference modelling | Planned |
| Hybrid Recommendation | Combined content and collaborative ranking | Planned |
