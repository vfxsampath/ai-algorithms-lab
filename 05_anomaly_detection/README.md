# Anomaly-Detection Algorithms

Anomaly-detection algorithms identify unusual observations that differ from the expected data distribution.

## Available Implementations

| Algorithm | Application | Status |
|---|---|---|
| [Isolation Forest](isolation_forest/) | Unsupervised operational anomaly detection | Complete |
| Local Outlier Factor | Local-density anomaly detection | Planned |
| One-Class SVM | Boundary-based novelty detection | Planned |
| Autoencoder | Neural reconstruction-based detection | Planned |
