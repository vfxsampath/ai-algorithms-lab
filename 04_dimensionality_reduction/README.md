# Dimensionality-Reduction Algorithms

Dimensionality-reduction techniques represent data using fewer variables while attempting to preserve important structure.

## Available Implementations

| Algorithm | Application | Status |
|---|---|---|
| [Principal Component Analysis](pca/) | Linear feature compression and visualization | Complete |
| Linear Discriminant Analysis | Supervised class-separation projection | Planned |
| t-SNE | Nonlinear visualization | Planned |
| UMAP | Nonlinear dimensionality reduction | Planned |
