# Clustering Algorithms

Clustering algorithms discover meaningful groups within unlabeled data.

## Available Implementations

| Algorithm | Business Use Case | Status |
|---|---|---|
| [K-Means](kmeans/) | Centroid-based product segmentation | Complete |
| [DBSCAN](dbscan/) | Density-based segmentation and noise detection | Complete |
| Hierarchical Clustering | Hierarchical product or customer grouping | Planned |
| HDBSCAN | Variable-density clustering | Planned |
| Gaussian Mixture Model | Probabilistic segmentation | Planned |
